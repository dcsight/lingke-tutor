# 英语答题与写作指南（领科 / WLSA 备考）

## 一、阅读理解

### 解题步骤
1. **先读题，后读文**：带着问题读，节省时间
2. **定位关键词**：把题目中的名词/数字在原文中定位
3. **排除干扰项**：
   - 过度推断（文中没说的结论）
   - 偷换概念（用近义词替换但意思变了）
   - 绝对化表达（always, never, only）要格外小心

### 常见题型
| 题型 | 策略 |
|------|------|
| 主旨题 | 关注首段和尾段，找作者的核心观点 |
| 细节题 | 回原文定位，逐字比对 |
| 推断题 | 只能推断文中"暗示"的，不能超出文本范围 |
| 词义猜测 | 利用上下文语境，而非字典含义 |
| 态度题 | 注意作者用词的情感色彩（positive/negative/neutral） |

## 二、语法与语言点

### 高频考点
- **时态一致**：叙述历史事件用一般过去时；表客观规律用一般现在时
- **主谓一致**：注意 "either...or / neither...nor" 就近原则
- **非谓语动词**：doing（主动/进行）、done（被动/完成）、to do（目的/将来）
- **从句**：定语从句 that/which/who；名词性从句 whether/that；状语从句连词

### 易错点
- "Despite / In spite of" 后接名词，不接从句（从句用 Although）
- "Make / Have / Let" 后接动词原形（使役动词）
- 比较结构 "more...than" 注意比较对象的一致性

## 三、议论文写作（Essay Writing）

### PEEL 结构（每段必用）
```
P - Point      | 本段论点（主题句）
E - Evidence   | 举例 / 数据 / 引用支撑
E - Explain    | 解释证据如何支持论点
L - Link       | 链接回文章总论点，过渡到下段
```

### 标准五段式框架
```
段1 Introduction
  - Hook（引题句）
  - Background（背景信息）
  - Thesis Statement（明确立场，一句话）

段2 Body 1（最强论点）
  - PEEL 结构

段3 Body 2（次要论点）
  - PEEL 结构

段4 Counterargument + Rebuttal（反驳段，可选）
  - 承认对立观点，再反驳

段5 Conclusion
  - Restate thesis（换说法重申立场）
  - Summary of arguments
  - Closing thought（展望或呼吁）
```

### 常用过渡词
| 功能 | 表达 |
|------|------|
| 添加论点 | Furthermore, Moreover, In addition |
| 举例 | For instance, To illustrate, As evidenced by |
| 转折 | However, Nevertheless, On the other hand |
| 因果 | Therefore, Consequently, As a result |
| 总结 | In conclusion, To sum up, Overall |

### 评分维度
1. **内容**（Ideas）：论点是否清晰、有说服力
2. **结构**（Organization）：段落逻辑是否连贯
3. **语言**（Language）：词汇是否多样，句式是否变化
4. **准确性**（Accuracy）：语法、拼写错误数量

## 四、面试英语（WLSA 专项）

### 自我介绍模板（90 秒）
```
Good morning/afternoon. My name is [Name], and I'm currently a 9th grader at [School].
I have a strong interest in [Subject/Field], especially [specific topic].
Outside of academics, I enjoy [hobby], which has taught me [skill/value].
I'm excited about the opportunity to join WLSA because [specific reason].
```

### 常见面试话题 & 答题框架
| 话题 | 策略 |
|------|------|
| 时事热点 | 表态 → 两点理由 → 一点反思（不要只说正面） |
| 个人经历 | STAR 法则：Situation / Task / Action / Result |
| 哲学/观点题 | 先定义核心概念，再给出有层次的分析 |

### 高分要素
- 主动使用学术词汇（utilize, analyze, perspective, implication）
- 说完一个观点后主动追问自己"However, one might argue..."
- 保持眼神接触，语速适中，停顿有力
